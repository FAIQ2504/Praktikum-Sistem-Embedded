## 🎯 Tujuan Praktikum

1. Mahasiswa mampu memahami konsep Analog-to-Digital Converter
2. Mahasiswa mampu mengimplementasikan program pada mikrokontroler
3. Mahasiswa mampu melakukan analisis hasil praktikum

## 📝 Tugas Praktikum

### Tugas 1: Implementasi Program Dasar
- Jalankan program pertama pada STM32/ESP32
- Dokumentasikan hasil output
- Analisis cara kerja program

### Tugas 2: Modifikasi Program
- Modifikasi salah satu parameter program
- Amati dan dokumentasikan perubahan
- Jelaskan alasan perubahan tersebut

### Tugas 3: Pengembangan
- Kembangkan program sesuai kreativitas
- Dokumentasikan dalam laporan
